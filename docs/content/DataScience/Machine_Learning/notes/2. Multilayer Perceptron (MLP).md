A **Multilayer Perceptron (MLP)** is a type of **feedforward artificial neural network** and one of the simplest forms of deep learning models. It consists of multiple layers of nodes (also called neurons), where each layer is **fully connected**to the next.

---

### 🔧 **Basic Architecture of an MLP:**

```
Input Layer → Hidden Layer(s) → Output Layer
```

Each layer consists of several neurons. Here's a breakdown of the components:

---

### 🧠 **1. Input Layer**

- Takes the raw input features.
    
- Number of neurons = number of input features (e.g., if your input is a vector of 10 features, this layer will have 10 neurons).
    

---

### 🏗️ **2. Hidden Layer(s)**

- One or more layers between input and output.
    
- Each neuron receives input from all neurons in the previous layer (fully connected).
    
- Each neuron computes:
    
    $z=W⋅x+ba=ϕ(z)$
    - W = weights
        
    - b = bias
        
    - ϕ = activation function (ReLU, sigmoid, tanh, etc.)
        
- These layers capture **nonlinear patterns** in data.
    

---

### 📤 **3. Output Layer**

- Produces final prediction.
    
- Number of neurons:
    
    - 1 for regression
        
    - Equal to number of classes for classification
        
- Activation function:
    
    - **None** or **linear** for regression
        
    - **Softmax** for multi-class classification
        
    - **Sigmoid** for binary classification
        

---

### 🔁 **Training (Backpropagation)**

- MLPs are trained using **backpropagation** and an optimization algorithm like **gradient descent** (often with variants like Adam).
    
- The model minimizes a **loss function** (MSE, cross-entropy, etc.) by updating weights using gradients.
    

---

### 📊 **Example MLP for Classification:**

- Input: 4 features
    
- Hidden layer 1: 8 neurons (ReLU)
    
- Hidden layer 2: 6 neurons (ReLU)
    
- Output: 3 classes (Softmax)
    

```
Input (4) → Dense(8, ReLU) → Dense(6, ReLU) → Dense(3, Softmax)
```

---

### ✅ Key Properties

- **Feedforward**: No cycles, data moves from input to output in one direction.
    
- **Fully connected**: Each neuron is connected to every neuron in the next layer.
    
- **Universal approximation**: With enough neurons and layers, MLP can approximate any continuous function.
