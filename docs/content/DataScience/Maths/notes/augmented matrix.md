### 📝 **Augmented Matrix**

An **augmented matrix** is a special matrix used to represent a system of linear equations. It combines the coefficient matrix and the constants from the right-hand side of the equations into a single matrix. The vertical bar `|` separates the coefficients (left side) from the constants (right side).

---

#### **How to Form an Augmented Matrix:**
Given a system of linear equations:
```
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
...
aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ
```

The **augmented matrix** is written as:
```
[ a₁₁  a₁₂  ...  a₁ₙ | b₁ ]
[ a₂₁  a₂₂  ...  a₂ₙ | b₂ ]
[ ...                ... ]
[ aₘ₁  aₘ₂  ...  aₘₙ | bₘ ]
```

---

#### **Example:**

Consider the system of equations:
```
x + 2y = 5
3x + 4y = 11
```

The augmented matrix would be:
```
[ 1  2 |  5 ]
[ 3  4 | 11 ]
```

---

#### **Purpose:**
The augmented matrix allows you to apply row operations (like those used in **[[Gaussian Elimination]]**) to simplify and solve the system of equations more efficiently.
