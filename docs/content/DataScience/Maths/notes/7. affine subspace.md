An affine subspace is essentially a vector subspace that has been shifted away from the origin (0,0)

An affine subspace is simply a vector subspace that has been shifted away from the origin. It's no longer a true vector subspace because it breaks the most important rule: it doesn't contain the neutral element.

Think of it this way:

- A **vector subspace** is a line or plane that is "anchored" to the origin.
    
- An **affine subspace** is that same line or plane just "floating" somewhere else in the space.
    

 `L = v + U`, describes this perfectly:

- `U` is the original vector subspace (the line or plane through the origin). This is called the **direction space**.
    
- `v` is the **support point**, which is the vector that shifts `U` away from the origin.
