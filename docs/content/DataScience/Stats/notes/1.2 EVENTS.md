- A collection or subset of possible [[1. THEORY OF PROBABILITY#^2d0716|outcomes]] having distinct or specific characteristics
- **Elementary event** an event which cannot be further divided into simpler events
- **composite event** a collection of event
- Let's break this down clearly, using your example of rolling two dice:

---
### example
#### **Composite Event: Total is 8**  
The event "total = 8" is a **composite event** because multiple combinations of dice rolls add up to 8.  
The possible outcomes are:  
- (2, 6)  
- (3, 5)  
- (4, 4)  
- (5, 3)  
- (6, 2)  

Each of these pairs is an **elementary event**, but together they form the composite event "total = 8."  

---

#### **Elementary Event: Total is 12**  
The event "total = 12" is an **elementary event** because there is only one possible outcome:  
- (6, 6)  

This cannot be decomposed further since it involves a single specific outcome.  

---

#### **Key Points**  
- A **composite event** is made up of multiple elementary events. For "total = 8," there are five specific outcomes contributing to the total, so it’s composite.  
- An **elementary event** involves a single, indivisible outcome. For "total = 12," there is only one possible outcome, so it’s elementary.  

---

#### **More Simple Examples**

#### **Example 1: Tossing Two Coins**  
- **Composite Event:** "Getting at least one Head" → Outcomes: {HH, HT, TH}  
- **Elementary Event:** "Getting both Tails" → Outcome: {TT}  

#### **Example 2: Rolling Two Dice**  
- **Composite Event:** "Total is 7" → Outcomes: {(1, 6), (2, 5), (3, 4), (4, 3), (5, 2), (6, 1)}  
- **Elementary Event:** "Total is 2" → Outcome: {(1, 1)}  

#### **Example 3: Picking a Card**  
- **Composite Event:** "Drawing a red card" → Outcomes: {Hearts and Diamonds (26 cards)}  
- **Elementary Event:** "Drawing the Ace of Spades" → Outcome: {Ace of Spades}
