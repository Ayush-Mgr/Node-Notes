### 🔹 **Indexing**

Used to access individual elements by their position.

**Python indexing starts from 0**

**Example (List):**

```python
fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple
print(fruits[2])    # cherry
print(fruits[-1])   # cherry (negative indexing)
```

**Example (NumPy):**

```python
import numpy as np
arr = np.array([10, 20, 30])
print(arr[1])       # 20
```

### 🔹 **Slicing**

Used to get a **sub-part** of the list/array.

**Syntax:** `list[start:stop:step]`

- `start` = index to start from (inclusive)
    
- `stop` = index to stop before (exclusive)
    
- `step` = how many steps to skip
    

**Examples:**

```python
numbers = [0, 1, 2, 3, 4, 5]

print(numbers[1:4])      # [1, 2, 3] - list[start:stop:step]
print(numbers[:3])       # [0, 1, 2]
print(numbers[3:])       # [3, 4, 5]
print(numbers[::2])      # [0, 2, 4]
print(numbers[::-1])     # [5, 4, 3, 2, 1, 0] (reverse)
```

**Example (NumPy 2D array):**

```python

# arr[row_start:row_stop:row_step, col_start:col_stop:col_step]

arr = np.array([[1, 2, 3], [4, 5, 6]])
print(arr[0, 1])     # 2 (row 0, column 1) 
print(arr[:, 1])     # [2 5] (all rows, column 1)

```
