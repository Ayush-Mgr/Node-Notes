## **Handling Missing Values in Pandas**

Missing data is often represented as **`NaN`** (Not a Number) in Pandas. You can **detect**, **remove**, or **fill** these values.

---

### 🔹 **1. Detect Missing Values**

```python
df.isnull()        # Returns True for NaN
df.isnull().sum()  # Count missing values per column
```

---

### 🔹 **2. Drop Missing Values**

- Drop rows:
    

```python
df.dropna(inplace=True)
```

- Drop columns:
    

```python
df.dropna(axis=1, inplace=True)
```

---

### 🔹 **3. Fill Missing Values**

- Fill with a specific value:
    

```python
df.fillna(0)
```

- Fill with mean/median/mode:
    

```python
df['Marks'].fillna(df['Marks'].mean(), inplace=True)
```

---

### 🔹 **4. Forward / Backward Fill**

```python
df.fillna(method='ffill')  # Forward fill
df.fillna(method='bfill')  # Backward fill
```

---

### ✅ **Example**

```python
import pandas as pd
data = {'Name': ['Alice', 'Bob', 'Charlie'], 'Score': [90, None, 85]}
df = pd.DataFrame(data)

df['Score'].fillna(df['Score'].mean(), inplace=True)
```
