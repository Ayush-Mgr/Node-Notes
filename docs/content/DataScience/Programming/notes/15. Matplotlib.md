## 🔹 **Introduction**

- **Matplotlib** is a popular Python library for creating **2D plots and graphs**.
    
- It helps visualize data through **line plots, bar charts, histograms, scatter plots**, and more.
    
- The main module used is `matplotlib.pyplot` (commonly imported as `plt`).
    

---

## 🔹 **Plotting using Matplotlib**

### Basic Steps:

1. Import the library.
    
2. Prepare your data.
    
3. Use plotting functions like `plt.plot()`.
    
4. Show the plot with `plt.show()`.
    

---

### Example: Simple Line Plot

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

plt.plot(x, y)       # Plot x vs y
plt.title('Simple Line Plot')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.show()
```

---

### Other Common Plots:

- **Bar chart:** `plt.bar(x, height)`
    
- **Histogram:** `plt.hist(data, bins=number)`
    
- **Scatter plot:** `plt.scatter(x, y)`
    

---
## 🔹 **Customization of Plots in Matplotlib**

You can customize plots by adding titles, labels, legends, colors, grid, and styles.

### Common Customizations:

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y, color='green', marker='o', linestyle='--', linewidth=2, markersize=8)
plt.title('Customized Plot', fontsize=14, fontweight='bold')
plt.xlabel('X Axis Label')
plt.ylabel('Y Axis Label')
plt.grid(True)
plt.legend(['Line 1'])
plt.show()
```
