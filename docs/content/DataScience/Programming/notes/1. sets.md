
### **MODULE 1: Introduction to Discrete Mathematics**

#### **Topic 1.1: Sets, Set Operations, and Venn Diagrams**

### **1. Concept Overview**

Aô **Set**ô is the fundamental building block of discrete math. In Data Science, think of a set as a collection of unique data points without any specific order. If you have a list of customer IDs with duplicates removed, that is a set.


### **3. Formal Definitions**

- **Set:**ô A well-defined collection of distinct objects.
    
    - Notation:ô $A = \{1, 2, 3\}$ô orô $S = \{x \mid x \in \mathbb{Z}^+, x < 5\}$ô (Set-builder notation: "The set of allô $x$ô such thatô $x$ô is a positive integer less than 5").
        
- **Element:**ô An object inside a set, denoted byô $\in$. ($1 \in A$).
    
- **Cardinality:**ô The number of elements in a set, denoted byô $|A|$. Ifô $A = \{a, b\}$, thenô $|A| = 2$.
    
- **Subset ($\subseteq$):**ô Setô $A$ô is a subset ofô $B$ô ifô **every**ô element inô $A$ô is also inô $B$.
    
- **Empty Set ($\emptyset$):**ô A set with no elements. Note:ô $|\emptyset| = 0$.
    

### **4. Set Operations (The Toolkit)**

Letô $U$ô be the Universal Set (the box containing everything relevant).

|**Operation**|**Symbol**|**Intuition**|**Formal Def**|
|---|---|---|---|
|**Union**|$A \cup B$|Combine both bags. (Logicalô **OR**)|$\{x \mid x \in A \lor x \in B\}$|
|**Intersection**|$A \cap B$|What do they share? (Logicalô **AND**)|$\{x \mid x \in A \land x \in B\}$|
|**Difference**|$A - B$ô orô $A \setminus B$|In A, butô **remove**ô anything that is also in B.|$\{x \mid x \in A \land x \notin B\}$|
|**Complement**|$A^c$ô orô $\bar{A}$|Everything in the Universe that isô **NOT**in A.|$\{x \mid x \in U \land x \notin A\}$|

### **5. Worked Examples**

**Example 1: Basic Operations (Easy)**

- **Given:**ô $U = \{1, 2, 3, 4, 5, 6\}$,ô $A = \{1, 2, 3\}$,ô $B = \{3, 4, 5\}$.
    
- **Find:**ô $A \cup B$,ô $A \cap B$, andô $A - B$.
    
- **Solution:**
    
    - $A \cup B = \{1, 2, 3, 4, 5\}$ô (Combine all unique numbers).
        
    - $A \cap B = \{3\}$ô (Only 3 is in both).
        
    - $A - B = \{1, 2\}$ô (Start with A, remove 3 because it's in B).
        

**Example 2: Cardinality and Subsets (Medium)**

- **Given:**ô Setô $S = \{a, \{a\}\}$.
    
- **Question:**ô What isô $|S|$? Isô $\{a\} \subseteq S$? Isô $\{a\} \in S$?
    
- **Solution:**
    
    - $|S| = 2$. The elements are the letter '$a$' and the set '$\{a\}$'.
        
    - $\{a\} \subseteq S$?ô **Yes.**ô The element '$a$' is insideô $S$.
        
    - $\{a\} \in S$?ô **Yes.**ô The set '$\{a\}$' is literally an element insideô $S$.
        
    - _Note: This is a common "trick" question in exams._
        

**Example 3: Set Builder Notation (Challenging)**

- **Given:**ô $A = \{x \in \mathbb{Z} \mid x^2 < 10\}$.
    
- **Task:**ô List the elements of A.
    
- **Solution:**
    
    - We need integers ($...-2, -1, 0, 1, 2...$) whose square is less than 10.
        
    - $0^2=0$,ô $1^2=1$,ô $2^2=4$,ô $3^2=9$,ô $4^2=16$ô (Stop).
        
    - Don't forget negatives!ô $(-1)^2=1$,ô $(-2)^2=4$,ô $(-3)^2=9$.
        
    - **Result:**ô $A = \{-3, -2, -1, 0, 1, 2, 3\}$.
        

### **6. Common Mistakes**

- **$\in$ô vsô $\subseteq$:**
    
    - $x \in A$ô meansô $x$ô is an item inside the bag.
        
    - $\{x\} \subseteq A$ô means the bag containingô $x$ô is inside (or equal to) bagô $A$.
        
- **The Empty Set:**
    
    - $\emptyset \subseteq A$ô isô **always true**ô for any setô $A$.
        
    - $\{\emptyset\}$ô is NOT empty. It is a bag containing an empty bag. Its cardinality is 1.
