## Grouped
- **Mean(Arithmetic Average):**
	- *[[continuous series]]
		- Sum of all data values divided by the number of data values.
	  
		- Formula: $$ \text{Mean} (\bar{x}) = \frac{\sum_{i=1}^{k} f_i m_i}{\sum_{i=1}^{k} f_i}
		  $$
		- $m$ = Upper Value  - Lower value / 2
		- **$fi$** = frequency of the `i` -th class,
		- **$mi$** = midpoint of the `i` -th class.

## Un Grouped
- **Mean (Arithmetic Average):**
	
	- Sum of all data values divided by the number of data values.
	- Formula: $$\text{Mean} (\bar{x}) = \frac{\sum_{i=1}^{n} x_i}{n}$$
	- where $x_i​$ is each individual data point, and $n$ is the total number of data points.
