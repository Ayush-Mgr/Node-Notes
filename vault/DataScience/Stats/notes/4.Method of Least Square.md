- Least square it checks how close to fitting your line( $y =a-bx_1$)  is to the data 
	- Least Square equation *Sum of squared errors* (SEE) = $\sum(y-\hat{y})^2$
		-  $\sum(y-\hat{y})^2$ = $\sum(y-a-bx_i)^2$  (since $\hat{y}=a-bx_1$ ,predicted value )
		- residuals $y-\hat{y}$ , distance between the actual data and  predicted data 
## Steps to perform Least Square method 
1. Draw a line which is most fitting to the data according to your judgment
2. Calculate estimated Y value or $\hat{y}$
3. Find the difference/ residuals  in the actual Y value and estimated Y value. $y_i - \hat{y_i}$ or $y-a-bx_i$
4. Square them and sum it.$\sum(y_i - \hat{y_i})^2$ or $\Sigma(y_i - a - bx_i)^2$
5. repeat 1, 2,3, 4 until you find least squared line

| Observed $x$ | Observed $y$ | Estimated $\hat{y}$    | Difference / residuals | Squared Difference         |
| ------------ | ------------ | ---------------------- | ---------------------- | -------------------------- |
| $x_i$        | $y_i$        | $\hat{y_i} = a + bx_i$ | $y_i - \hat{y_i}$      | $(y_i - \hat{y_i})^2$      |
| $x_1$        | $y_1$        | $a + bx_1$             | $y_1 - a - bx_1$       | $(y_1 - a - bx_1)^2$       |
| $x_2$        | $y_2$        | $a + bx_2$             | $y_2 - a - bx_2$       | $(y_2 - a - bx_2)^2$       |
| ...          | ...          | ...                    | ...                    | ...                        |
| $x_n$        | $y_n$        | $a + bx_n$             | $y_n - a - bx_n$       | $(y_n - a - bx_n)^2$       |
| **Total**    |              |                        |                        | $\Sigma(y_i - a - bx_i)^2$ |
