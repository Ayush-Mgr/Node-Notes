### 🔹 **Introduction to Data**

**Data** is raw information — numbers, text, images, etc. — collected for analysis.

**Types of Data:**

- **Qualitative** (e.g., colors, names)
    
- **Quantitative** (e.g., age, height)
    

**Example:**

```python
name = "John"
age = 25
```

---

### 🔹 **Data Collection**

Gathering data from different sources.

**Methods:**

- Surveys
    
- Sensors
    
- Online forms
    
- Databases
    
- Web scraping
    

**Example:**

```python
response = input("Enter your age: ")
```

---

### 🔹 **Data Storage**

Saving collected data for future use.

**Methods:**

- Files (CSV, TXT, JSON)
    
- Databases (MySQL, MongoDB)
    
- Cloud storage (Google Drive, AWS)
    

**Example:**

```python
with open("data.txt", "w") as file:
    file.write("Name: John, Age: 25")
```

---

### 🔹 **Data Processing**

Cleaning, organizing, and converting data into usable format.

**Steps:**

- Remove duplicates
    
- Fix errors
    
- Convert types
    
- Sort or filter data
    

**Example (Python):**

```python
data = ["10", "20", "30"]
data = [int(i) for i in data]
print(sum(data))   # 60
```

---

### 🔹 **Statistical Techniques for Data Processing**

Used to analyze and interpret data.

**Common Techniques:**

- **Mean** (average)
    
- **Median** (middle value)
    
- **Mode** (most frequent)
    
- **Standard deviation** (data spread)
    

**Example:**

```python
from scipy import stats  
import numpy as np
nums = [10, 20, 30, 20]

nums= np.array(nums)

print(np.mean(nums))  # 20
print(np.median(nums)) # 20.0
print(stats.mode(nums))   # 20
```
