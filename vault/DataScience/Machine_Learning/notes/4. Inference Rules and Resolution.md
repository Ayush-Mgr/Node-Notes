# Inference Rules and Resolution
---
**Inference Rules** help us to get conclusion on given [[1 . Propositional logic|arguments]] using some principles which are:

### **Modus Ponens (Law of Detachment)**  
If $p \rightarrow q$ and $p$ are true, then $q$ is true.

```
p -> q  
p  
-------  
∴ q
```

**Example**:  
- If it rains, the ground will be wet.  
- It rains.  
- **Conclusion**: The ground is wet.

---

### **Modus Tollens (Denying the Consequent)**  
If $p \rightarrow q$ and $q$ is false, then $p$ is false.

```
p -> q  
~q  
-------  
∴ ~p
```

**Example**:  
- If it rains, the ground will be wet.  
- The ground is not wet.  
- **Conclusion**: It is not raining.

---

### **Hypothetical Syllogism**  
If $p \rightarrow q$ and $q \rightarrow r$, then $p \rightarrow r$.

```
p -> q  
q -> r  
-------  
∴ p -> r
```

**Example**:  
- If I study, I will pass the exam.  
- If I pass the exam, I will graduate.  
- **Conclusion**: If I study, I will graduate.

---

### **Disjunctive Syllogism**  
If $p \lor q$ and $p$ is false, then $q$ is true.

```
p v q  
~p  
-------  
∴ q
```

**Example**:  
- It is either raining or sunny.  
- It is not raining.  
- **Conclusion**: It is sunny.

---

### **Simplification**  
If $p \land q$, then $p$ is true (and $q$ is also true).

```
p ∧ q  
-------  
∴ p
```

**Example**:  
- It is raining and cold.  
- **Conclusion**: It is raining.

---

### **Conjunction**  
If $p$ and $q$ are true, then $p \land q$ is true.

```
p  
q  
-------  
∴ p ∧ q
```

**Example**:  
- It is raining.  
- It is cold.  
- **Conclusion**: It is raining and cold.

---

### **Addition**  
If $p$ is true, then $p \lor q$ is true.

```
p  
-------  
∴ p v q
```

**Example**:  
- It is raining.  
- **Conclusion**: It is raining or sunny.

---

### **Resolution**  
If $p \lor q$ and $\neg p \lor r$, then $q \lor r$ is true.

```
p v q  
~p v r  
-------  
∴ q v r
```

**Example**:  
- It is raining or cloudy ($p \lor q$).  
- It is not raining or windy ($\neg p \lor r$).  
- **Conclusion**: It is cloudy or windy ($q \lor r$).
