### 1.**Feedforward Neural Network (FNN)**

- **Structure:** Data flows in one direction, from input to output.
    
- **Use cases:** Image recognition, regression tasks, simple classification.
    
- **Example architectures:** Multi-layer Perceptron (MLP).
----

### 2.**Recurrent Neural Network (RNN)**

- **Structure:** Has loops to allow information to persist over time.
    
- **Use cases:** Sequence prediction, natural language processing (NLP), time series forecasting.
    
- **Limitations:** Struggles with long-term dependencies.
