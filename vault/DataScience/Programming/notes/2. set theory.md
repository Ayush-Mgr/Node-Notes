
### **1. Concept Overview**

Just like in algebra where $x(y+z) = xy + xz$, sets have rules that allow you to transform and simplify expressions. In Data Science, you use these laws constantly to optimize database queries or clean logic in code (e.g., simplifying `if`statements).

### **2. Intuition (Algebra Analogy)**

Think of Union ($\cup$) like **Addition** (+) and Intersection ($\cap$) like **Multiplication** ($\times$).

- **Distributive Law:** In math, $3 \times (4 + 5) = (3 \times 4) + (3 \times 5)$.
    
- In sets, $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$.
    

### **3. Formal Definitions (The Laws)**

You must memorize these standard laws. $U$ is the universal set, $\emptyset$ is the empty set.

| **Law Name**       | **Rule**                                                                                                           |
| ------------------ | ------------------------------------------------------------------------------------------------------------------ |
| **Idempotent**     | $A \cup A = A$<br><br>  <br><br>$A \cap A = A$                                                                     |
| **Identity**       | $A \cup \emptyset = A$<br><br>  <br><br>$A \cap U = A$                                                             |
| **Domination**     | $A \cup U = U$<br><br>  <br><br>$A \cap \emptyset = \emptyset$                                                     |
| **Complement**     | $A \cup A^c = U$<br><br>  <br><br>$A \cap A^c = \emptyset$                                                         |
| **Involution**     | $(A^c)^c = A$                                                                                                      |
| **Commutative**    | $A \cup B = B \cup A$<br><br>  <br><br>$A \cap B = B \cap A$                                                       |
| **Associative**    | $(A \cup B) \cup C = A \cup (B \cup C)$                                                                            |
| **Distributive** ⭐ | $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$<br><br>  <br><br>$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$ |
| **De Morgan’s** ⭐  | $(A \cup B)^c = A^c \cap B^c$<br><br>  <br><br>$(A \cap B)^c = A^c \cup B^c$                                       |
| **Absorption**     | $A \cup (A \cap B) = A$<br><br>  <br><br>$A \cap (A \cup B) = A$                                                   |

> **Note on De Morgan's:** "Break the bar, change the sign." When you distribute the complement (the "bar") inside the parentheses, the Union flips to Intersection, and vice versa.

### **4. Worked Examples**

**Example 1: Verification (Easy)**

- **Given:** $U=\{1,2,3,4\}$, $A=\{1,2\}$, $B=\{2,3\}$.
    
- **Verify De Morgan's Law:** $(A \cup B)^c = A^c \cap B^c$.
    
- **LHS (Left Hand Side):**
    
    - $A \cup B = \{1, 2, 3\}$
        
    - $(A \cup B)^c = \{4\}$ (Everything in U except 1, 2, 3)
        
- **RHS (Right Hand Side):**
    
    - $A^c = \{3, 4\}$
        
    - $B^c = \{1, 4\}$
        
    - $A^c \cap B^c = \{4\}$
        
- **Result:** LHS = RHS. Verified.
    

**Example 2: Simplification (Medium - Exam Favorite)**

- **Simplify:** $(A \cap B) \cup (A \cap B^c)$
    
- **Step 1:** Notice "A" is in both parts. Use **Distributive Law in reverse**.
    
    - Factor out $A \cap$:
        
    - $A \cap (B \cup B^c)$
        
- **Step 2:** Look at $(B \cup B^c)$. Use **Complement Law**.
    
    - $B \cup B^c = U$
        
    - Expression becomes: $A \cap U$
        
- **Step 3:** Use **Identity Law**.
    
    - $A \cap U = A$
        
- **Final Answer:** $A$
    

**Example 3: Algebraic Proof (Challenging)**

- **Prove:** $A - B = A \cap B^c$
    
- **Logic:**
    
    - $x \in A - B$ means $x$ is in $A$ **AND** $x$ is **NOT** in $B$.
        
    - "Not in $B$" is the definition of $B^c$.
        
    - "AND" is the definition of Intersection $\cap$.
        
    - Therefore, $A - B$ is equivalent to $A \cap B^c$.
        
    - _Tip: You can use this identity to simplify differences in proofs!_
        

### **5. Common Mistakes**

- **Forgetting to flip the sign** in De Morgan's Law: $(A \cup B)^c \neq A^c \cup B^c$. It must become $\cap$.
    
- **Distributing incorrectly:** $A \cup (B \cap C) \neq (A \cup B) \cap C$. You must distribute $A$ to **both** terms: $(A \cup B) \cap (A \cup C)$.
    

### **6. Data Science Connection**

- **Boolean Logic in Pandas/NumPy:**
    
    - When filtering data: `df[~( (df.age > 50) | (df.salary > 100k) )]`
        
    - Using De Morgan's, this is the same as: `df[ (df.age <= 50) & (df.salary <= 100k) ]`
        
    - The second version is often easier to read and debug.
