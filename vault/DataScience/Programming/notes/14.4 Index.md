## **Altering the Index in Pandas**

You can change (or set) the index of a DataFrame using several methods:

---

### 🔹 **1. Set a Column as Index**

```python
df.set_index('column_name', inplace=True)
```

### 🔹 **2. Reset Index to Default (0, 1, 2...)**

```python
df.reset_index(inplace=True)
```

### 🔹 **3. Rename Index**

```python
df.index = ['a', 'b', 'c']  # Manually assign new index
```

---

### ✅ **Example:**

```python
import pandas as pd

data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)

# Set 'Name' as index
df.set_index('Name', inplace=True)
```
