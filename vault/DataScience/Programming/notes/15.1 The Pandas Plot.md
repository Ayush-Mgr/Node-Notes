
## 🔹 **The Pandas Plot Function (Pandas Visualization)**

- Pandas DataFrames and Series have a built-in `.plot()` method.
    
- It’s a simple way to create plots without importing Matplotlib explicitly.
    
- Supports line, bar, histogram, pie, scatter, etc.
    

### Example:

```python
import pandas as pd

data = {'Year': [2018, 2019, 2020, 2021],
        'Sales': [200, 300, 400, 500]}
df = pd.DataFrame(data)

df.plot(x='Year', y='Sales', kind='line', title='Sales Over Years', grid=True)
```

---

### Quick Notes:

|Parameter|Description|
|---|---|
|`kind`|Type of plot (`'line'`, `'bar'`, `'hist'`, `'scatter'`, etc.)|
|`title`|Plot title|
|`grid`|Show grid lines (`True`/`False`)|
|`xlabel`, `ylabel`|Axis labels (use Matplotlib functions if needed)|
