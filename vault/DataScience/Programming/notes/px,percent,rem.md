### Summary Cheat Sheet

| Unit                                     | Best Used For               | Result                                                                                            |
| ---------------------------------------- | --------------------------- | ------------------------------------------------------------------------------------------------- |
| **```px```**     | Cards, Buttons, Input boxes | **Symmetrical corners.** Always looks "round."                                                    |
| **```%```**      | Avatars, Icons              | **Circles.** Use ```50%``` on a square image to make it a perfect circle. |
| **```em/rem```** | Typography elements         | **Scalable.** Curves that grow if the font size grows.                                            |
