### 🔹 **What is Reshaping?**

**Reshaping** means changing the shape (rows × columns) of a NumPy array **without changing its data**.

---

### 🔹 **Syntax:**

```python
array.reshape(new_shape)
```

New shape must have **same total number of elements**.

---

### 🔹 **Example: 1D to 2D**

```python
import numpy as np

a = np.array([1, 2, 3, 4, 5, 6])

b = a.reshape(2, 3)
print(b)
# Output:
# [[1 2 3]
#  [4 5 6]]
```

---

### 🔹 **2D to 1D (Flattening)**

```python
b = np.array([[1, 2, 3], [4, 5, 6]])
c = b.reshape(-1)
print(c)     # [1 2 3 4 5 6]
```

---

### 🔹 **Using -1 for Auto Calculation**

NumPy will calculate the missing dimension automatically.

```python
a = np.array([1, 2, 3, 4, 5, 6])
print(a.reshape(3, -1))   # (3 rows, auto cols)
# Output:
# [[1 2]
#  [3 4]
#  [5 6]]
```

---

### ✅ Rules:

- Total elements must match: `original_size = new_shape[0] * new_shape[1] * ...` 
    
- You can use `-1` to auto-calculate one dimension.
