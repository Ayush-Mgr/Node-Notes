### 1. What is Semantic Analysis?

**Semantic Analysis** is the process of extracting **meaning** from text.

- **Syntactic Analysis** focuses on structure (grammar).
    
- **Semantic Analysis** goes deeper to understand the _contextual meaning_ of words, phrases, and sentences.
    

### 2. Word Sense Disambiguation (WSD)

**Definition:** WSD is the process of determining which sense (meaning) of a word is used in a particular context when the word has multiple meanings (**polysemy**).

**Example: The word "bank"** 

| Sentence                              | Meaning (Sense)       |
| :------------------------------------ | :-------------------- |
| "He sat by the river **bank**."       | Land beside a river   |
| "He deposited money in the **bank**." | Financial institution |
|                                       |                       |

#### Methods of WSD

| Method Type             | Description                                       | Example Tool     |
| ----------------------- | ------------------------------------------------- | ---------------- |
| **Dictionary-based**    | Uses definitions from WordNet; overlaps meanings. | Lesk Algorithm   |
| **Supervised Learning** | Uses labeled data with correct senses.            | SVM, Naive Bayes |
| **Unsupervised**        | Uses clustering to infer senses.                  | EM Algorithm     |
| **Knowledge-based**     | Uses ontologies or semantic networks.             | WordNet          |

**Use Cases:**

- **Machine Translation:** Distinguishing "bank" as _banque_ (finance) vs. _rive_ (river) in French.
    
- **Search Engines:** Knowing if "java" refers to the programming language or the island.
    
- **Chatbots:** Understanding commands with ambiguous words.
    

### 3. Named Entity Recognition (NER)

**Definition:** NER identifies and classifies named entities in text into predefined categories.

**Common Entity Types:** 
  
| Type           | Examples                    |
| -------------- | --------------------------- |
| **PERSON**     | Barack Obama, Elon Musk     |
| **ORG**        | Microsoft, WHO              |
| **GPE**        | India, New York (Locations) |
|  **TIME**      | 3 PM, noon                  |
|  **MONEY**     | $50, ₹1000                  |
| **PRODUCT**    | iPhone, Tesla               |

**Example:** _"Apple Inc. released the new iPhone in California on Tuesday."_

- **Apple Inc.** $\to$ ORGANIZATION
    
- **iPhone** $\to$ PRODUCT
    
- **California** $\to$ LOCATION
    
- **Tuesday** $\to$ DATE
    

### 4. Semantic Role Labeling (SRL)

**Definition:** SRL identifies the **predicate-argument structure** of a sentence. It answers:

- What is happening? (Predicate)
    
- Who is doing it? (Agent)
    
- To whom? (Theme/Recipient)
    
- When/Where? (Time/Location)
    

**Example:** _"John gave Mary a book at the library."_ 

| Role                 | Word           |
| :------------------- | :------------- |
|  **Predicate**       | gave           |
|  **Agent** (Doer)    | John           |
| **Theme** (Affected) | a book         |
| **Recipient**        | Mary           |
| **Location**         | at the library |

### 5. Summary of Tasks

| Task    | Goal                                 | Output Example                            |
| ------- | ------------------------------------ | ----------------------------------------- |
| **WSD** | Resolve meaning of polysemous words. | "bank" $\to$ financial institution        |
| **NER** | Extract & classify named entities.   | "Barack Obama" $\to$ PERSON               |
| **SRL** | Identify who did what to whom.       | "John gave Mary..." $\to$ Agent/Recipient |

### 6. Comparison: Syntactic vs. Semantic vs. Sentiment

|Type of Analysis|Definition|Goal|Techniques|
|---|---|---|---|
|**Syntactic Analysis**|Analyzes grammatical structure.|Understand structure; identify POS and phrase relationships.|POS Tagging, Parsing|
|**Semantic Analysis**|Understands meaning in context.|Determine intended meaning of words and their roles.|WSD, NER, SRL|
|**Sentiment Analysis**|Detects emotions/opinions.|Classify emotional tone (positive, negative, neutral).|Polarity detection|
