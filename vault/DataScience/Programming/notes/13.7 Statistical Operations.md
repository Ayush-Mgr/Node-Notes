### 🔹 **What Are Statistical Operations?**

They help you **analyze and summarize** data using stats like mean, median, min, max, etc.

You need:

```python
import numpy as np
```

---

### 🔹 **1. Mean (Average)**

```python
a = np.array([10, 20, 30])
print(np.mean(a))     # 20.0
```

---

### 🔹 **2. Median (Middle value)**

```python
a = np.array([10, 30, 20])
print(np.median(a))   # 20.0
```

---

### 🔹 **3. Mode (Most frequent value)**

For mode, use `scipy`:

```python
from scipy import stats
a = np.array([1, 2, 2, 3])
print(stats.mode(a, keepdims=True).mode[0])  # 2
```

---

### 🔹 **4. Min & Max**

```python
a = np.array([5, 10, 15])
print(np.min(a))      # 5
print(np.max(a))      # 15
```

---

### 🔹 **5. Standard Deviation (Spread of data)**

```python
a = np.array([1, 2, 3, 4])
print(np.std(a))      # 1.118...
```

---

### 🔹 **6. Sum & Product**

```python
a = np.array([1, 2, 3])
print(np.sum(a))      # 6
print(np.prod(a))     # 6
```

---

### 🔹 **7. Axis-Based Operations (for 2D arrays)**

```python
b = np.array([[1, 2], [3, 4]])

print(np.sum(b, axis=0))  # [4 6] (column-wise)
print(np.sum(b, axis=1))  # [3 7] (row-wise)
```
