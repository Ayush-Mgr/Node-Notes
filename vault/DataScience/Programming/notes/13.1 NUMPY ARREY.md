### 🔹 **Introduction to NumPy**

**NumPy** (Numerical Python) is a powerful Python library used for:

- Scientific computing
    
- Working with large multi-dimensional arrays and matrices
    
- Performing fast mathematical operations
    

You must install it first (if not already):

```bash
pip install numpy
```

Then import it:

```python
import numpy as np
```

---

### 🔹 **Array and NumPy Array**

#### ✅ **Regular Python List (Array-like):**

- Slower for large data
    
- No built-in math operations
    

**Example:**

```python
list1 = [1, 2, 3]
print(list1[0])       # Output: 1
```

#### ✅ **NumPy Array:**

- Faster, more efficient
    
- Supports element-wise operations
    

**Creating a NumPy array:**

```python
import numpy as np

arr = np.array([1, 2, 3])
print(arr)            # [1 2 3]
print(arr + 5)        # [6 7 8] (adds 5 to each element)
```

---

### 🔹 **Differences:**

|Feature|Python List|NumPy Array|
|---|---|---|
|Speed|Slower|Faster|
|Memory usage|More|Less|
|Element operations|Manual loops|Direct operations|
