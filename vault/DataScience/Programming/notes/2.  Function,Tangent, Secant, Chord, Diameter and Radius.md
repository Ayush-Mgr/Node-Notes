# Functions 

A **function** is a rule or operation that takes an input and produces an output.

Example:  
$$
f(x) = x^2
$$

- Input: $x$ (a number)  
- Operation: Square it ($x \times x$)  
- Output: $x^2$  

If we give the function many values of $x$, we can plot the outputs.  
For $f(x) = x^2$, this forms a **parabola**.

---

### Visual intuition
- Each input $x$ → gets mapped to a single output $f(x)$.  
- Together, the collection of $(x, f(x))$ points makes a curve.  
- For $f(x) = x^2$, the curve is U-shaped.

---
📌 Think of a function as a **machine**:  
Input → [Function Rule] → Output

# Geometry Basics: Circle Terms

## Tangent
- Touches the circle at **exactly one point**.  
- Perpendicular to the radius at that point.  

## Secant
- A **line** that cuts through the circle, intersecting it at **two points**.  
- Extends infinitely in both directions.  

## Chord
- A **line segment** that connects two points **on the circle**.  
- Unlike a secant, it **does not extend outside** the circle.  
- The **diameter** is the longest chord.  

## Radius
- A line from the center → to a point on the circle.  

## Diameter
- A chord passing through the center.  
- Equal to **2 × radius**.
