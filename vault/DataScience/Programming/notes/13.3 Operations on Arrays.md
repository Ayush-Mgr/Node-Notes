### 🔹 **1. Arithmetic Operations**

You can perform element-wise math on NumPy arrays (like add, subtract, etc.).

**Example:**

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print(a + b)     # [5 7 9]
print(a - b)     # [-3 -3 -3]
print(a * b)     # [4 10 18]
print(a / b)     # [0.25 0.4  0.5 ]
```

---

### 🔹 **2. Scalar Operations**

Apply an operation to every element using a single number.

**Example:**

```python
a = np.array([10, 20, 30])
print(a + 5)     # [15 25 35]
print(a * 2)     # [20 40 60]
```

---

### 🔹 **3. Comparison Operations**

Compare elements to a number or another array.

**Example:**

```python
a = np.array([1, 2, 3])
print(a > 1)     # [False  True  True]
```

---

### 🔹 **4. Aggregate Functions**

Get summary stats from arrays.

**Example:**

```python
a = np.array([1, 2, 3, 4])

print(np.sum(a))       # 10
print(np.mean(a))      # 2.5
print(np.min(a))       # 1
print(np.max(a))       # 4
print(np.std(a))       # 1.118...
```

---

### 🔹 **5. Matrix Operations (for 2D arrays)**

Used for linear algebra.

**Example:**

```python
A = np.array([[1, 2],
			  [3, 4]])
			  
B = np.array([[5, 6],
			  [7, 8]])

print(np.dot(A, B))
# Output:
# [[19 22]
#  [43 50]]
```
