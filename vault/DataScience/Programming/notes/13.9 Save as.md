### 🔹 **1. Save as `.npy` (Binary Format for One Array)**

Use `np.save()`

```python
import numpy as np

arr = np.array([10, 20, 30])
np.save('my_array.npy', arr)   # Saves in current folder
```

✅ To load it later:

```python
loaded = np.load('my_array.npy')
print(loaded)    # [10 20 30]
```

---

### 🔹 **2. Save Multiple Arrays as `.npz` (Compressed Format)**

Use `np.savez()`

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
np.savez('arrays_file.npz', first=a, second=b)
```

✅ To load:

```python
data = np.load('arrays_file.npz')
print(data['first'])    # [1 2 3]
print(data['second'])   # [4 5 6]
```

---

### 🔹 **3. Save as `.txt` or `.csv` (Human-readable format)**

Use `np.savetxt()`

```python
arr = np.array([[1, 2], [3, 4]])
np.savetxt('array_data.txt', arr)
```

✅ Save with comma separator (CSV format):

```python
np.savetxt('array_data.csv', arr, delimiter=',', fmt='%d')
```

---

### ✅ Summary Table

|Function|File Type|Description|
|---|---|---|
|`np.save()`|`.npy`|Saves a single array|
|`np.savez()`|`.npz`|Saves multiple arrays|
|`np.savetxt()`|`.txt`/`.csv`|Saves array in text format|
