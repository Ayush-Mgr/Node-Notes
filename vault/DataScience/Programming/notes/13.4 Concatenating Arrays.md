### 🔹 **What is Concatenation?**

**Concatenating arrays** means joining two or more arrays into one.

---

### ✅ **1. Concatenating 1D Arrays**

Use `np.concatenate()` to join arrays **along the same axis**.

**Example:**

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

c = np.concatenate((a, b))
print(c)      # [1 2 3 4 5 6]
```

---

### ✅ **2. Concatenating 2D Arrays**

You can join 2D arrays **vertically** (rows) or **horizontally** (columns).

**➤ Vertically (axis=0):**

```python
a = np.array([[1, 2], [3, 4]])
b = np.array([[5, 6]])

c = np.concatenate((a, b), axis=0)
print(c)
# [[1 2]
#  [3 4]
#  [5 6]]
```

**➤ Horizontally (axis=1):**

```python
a = np.array([[1, 2], [3, 4]])
b = np.array([[5], [6]])

c = np.concatenate((a, b), axis=1)
print(c)
# [[1 2 5]
#  [3 4 6]]
```

---

### ⚠️ **Note:** The dimensions must match **except** in the axis you’re joining on.
