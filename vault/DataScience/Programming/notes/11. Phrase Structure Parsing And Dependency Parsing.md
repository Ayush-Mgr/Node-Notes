- **POS Tagging** = Labeling the bricks (this is a red brick, this is a blue brick).
    
- **Phrase Parsing** = Showing how the bricks are cemented together to build a wall (this stack of bricks is a "Subject", that stack is a "Predicate").

### 1. Phrase Structure Parsing (Constituency Parsing)

**Definition:** Phrase Structure Parsing identifies the nested phrase structure of a sentence by breaking it down into **constituents** (phrases like Noun Phrases, Verb Phrases, etc.). It typically relies on rules from **Context-Free Grammar (CFG)**.

#### Common Phrase Types

|Phrase Type|Meaning|Example|
|---|---|---|
|**NP**|Noun Phrase|_the lazy dog_|
|**VP**|Verb Phrase|_eats an apple_|
|**PP**|Prepositional Phrase|_in the garden_|
|**ADJP**|Adjective Phrase|_very tall_|
|**ADVP**|Adverb Phrase|_quite slowly_|

#### Structural Example

**Sentence:** _"The dog barked loudly."_

**Tree Representation:**

```
(S
  (NP (DT The) (NN dog))
  (VP (VBD barked) 
      (ADVP (RB loudly)))
)
```

**Breakdown:**

- **NP (Noun Phrase):** "The dog"
    
- **VP (Verb Phrase):** "barked loudly"
    
- **ADVP (Adverb Phrase):** "loudly" (nested inside the VP)
    

### 2. Dependency Parsing

**Definition:** Dependency Parsing determines how words are related to each other using **head-dependent relations** (e.g., subject, object, modifier). Unlike phrase structure parsing, it connects words directly using **directed arcs** (Head $\to$Dependent).

#### Dependency Example

**Sentence:** _"The dog barked loudly."_

- **ROOT:** _barked_ (Main verb)
    
- **nsubj (Subject):** _dog_ $\to$ depends on _barked_
    
- **det (Determiner):** _The_ $\to$ depends on _dog_
    
- **advmod (Adverbial Modifier):** _loudly_ $\to$ depends on _barked_
    

**Visual Logic:**

```
      [ROOT]
         |
       barked
       /    \
     dog   loudly
      |
     The
```

#### Dependency Relation Types

|Relation|Meaning|
|---|---|
|**nsubj**|Nominal subject|
|**dobj**|Direct object|
|**amod**|Adjective modifier|
|**advmod**|Adverbial modifier|
|**det**|Determiner|
|**prep**|Prepositional modifier|
|**pobj**|Object of a preposition|
|**ROOT**|Root verb of the sentence|

### 3. Comparison: Phrase Structure vs. Dependency Parsing

|Feature|Phrase Structure Parsing|Dependency Parsing|
|---|---|---|
|**Structure Type**|Constituency tree (Nested phrases)|Dependency tree (Word-to-word links)|
|**Focus**|Phrase grouping (NP, VP)|Head-dependent word relations|
|**Root**|Sentence (S)|Main verb (ROOT)|
|**Used In**|Grammar-based NLP|Modern NLP (syntax, translation, extraction)|
|**Tools**|NLTK, Berkeley Parser|spaCy, Stanza, UDPipe|

### 4. Applications of Parsing

|Application|Usage|
|---|---|
|**Question Answering**|Understanding Subject-Verb-Object relationships to pinpoint answers.|
|**Machine Translation**|Essential for sentence reordering between languages with different grammar rules.|
|**Grammar Correction**|Identifying incorrect structures or broken dependencies.|
|**Information Extraction**|Extracting specific relations like _Subject-Action_.|
|**Semantic Role Labeling**|Understanding the underlying meaning and roles within a sentence.|
