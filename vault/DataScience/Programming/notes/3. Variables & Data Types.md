
**Variables:**  
Variables are used to store data values in Python and can change during program execution.


**Data Types:**  
Data types define the kind of value a variable holds, such as `int` (integer), `float` (decimal), `str` ([[string]]), `bool`(True/False), and more.

---

**Difference between Variables and [[2. keywords and identifiers#^043d84|Identifiers]]:**  
A **variable** is a name used to store data, while an **identifier** is the general name used for any user-defined element like variables, functions, or classes — so all variables are identifiers, but not all identifiers are variables.

```python
x = 10             # 'x' is a variable (and also an identifier)
def greet():       # 'greet' is an identifier (but it's a function, not a variable)
    pass
```
