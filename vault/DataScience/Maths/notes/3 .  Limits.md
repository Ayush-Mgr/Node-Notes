### What Each Statement Means

Suppose we were given the equation $f(x) = x² + 7x + 8$ forming a curve
and Limit:
$$
\lim_{x \to -6} f(x) = 2
$$


<img src ='Pasted image 20251008090139.png' style="filter: invert(1);">



1. **`lim x→-6 f(x) = 2` (The Limit)**
    $$
\lim_{x \to -6} f(x) = 2
$$
    - This statement describes the **approach** both side.
        
    - It means: "As the input `x` gets infinitely close to -6 (from both the left and right sides), the output value of the function `f(x)` gets infinitely close to 2."
        
    - This is the _expected_ value or the value the function is trending towards.
        
2. **`f(-6) = 2` (The Function Value)**
    
    - This statement describes the **actual point**.
        
    - It means: "When the input `x` is _exactly_ -6, the output of the function `f(x)` is _exactly_ 2."
        
    - This is the actual value of the point on the graph.
