##  a **generating set**
 A set of vectors is a generating set if you can use them to create _any other vector_ in the entire space through [[4. linear Independence|linear combinations]]

## span
The set of all vectors you can create is called the **span**.

eg,
using `E = (1,0)` and `N = (0,1)` vectors. We saw we could create any point `(x, y)` by combining them. This means the set `{E, N}` is a generating set that spans the entire 2D plane `R²`.

## Rank

 The **rank** of a matrix is simply the number of linearly independent rows (or columns) it has.

Think of it as a measure of the "non-redundant" information in the matrix. 

Let's look at the first example from your text:

```
A = [[1, 0, 1],
     [0, 1, 1],
     [0, 0, 0]]
```

here Rank is 2 or $rk(A) = 2$ because $3^{rd}$ row is a linearly dependent vector
