A **polynomial** is an algebraic expression made up of variables, coefficients, and exponents, combined using addition, subtraction, and multiplication (but no division by variables).

## Example 


$$
P(x) = x^2 + 2x + 3
$$

Let's break it down:  
- $x^2$  → Quadratic term (degree 2)  
- $2x$  → Linear term (degree 1)  
- $3$  → Constant term (degree 0)  

Now, let's try some basic operations:  

### **Addition Example**  
If we add another polynomial $Q(x) = x + 1$  to $P(x) = x^2 + 2x + 3$ :  
$$
(x^2 + 2x + 3) + (x + 1) = x^2 + 3x + 4
$$

### **Multiplication Example**  
If we multiply by $(x + 1)$ :  
$$
(x^2 + 2x + 3) \times (x + 1)
$$
Expanding using distribution:  
$$
x^3 + x^2 + 2x^2 + 2x + 3x + 3 = x^3 + 3x^2 + 5x + 3
$$
