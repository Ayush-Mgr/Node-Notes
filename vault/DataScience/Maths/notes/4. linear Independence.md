## Linear Combinations:

Vector  you get from some vectors, scale them (stretch or shrink them), performing any of the scaling or adding.

The key is that a linear combination is the **single final vector** you get after scaling and adding. If you scale two vectors but don't add them, you've just created two separate linear combinations

$$v_1*1 + v_2*2 = v_3$$

## Redundant Vectors

If you can create one vector in a set by making a linear combination of the others, that vector is redundant. Its presence in the set creates **linear dependence**

$$V1+V2 = V3 \text{ (V3 is Redundant) }$$
## Linear independence
A set of vectors is **linearly independent** if the only way to create a [[4. linear Independence|linear combination]] that equals the **zero vector** is by using the **zero scalar** (_neutral element_) for every vector.
$$V_1*0 + V_2*0 = 0$$
Or,
if the vectors set is not redundant, then they are linearly independent

## linearly dependent

>==any set of **three or more** vectors will always be linearly dependent in 2D space.==
>
>==any set of vector with  neutral value is as well linear dependency==

A set of vectors is **linearly dependent**  if there is many way to create a [[4. linear Independence|linear combination]] that equals the **zero vector** / neutral.
$$V_1*\lambda_1 + V_2*\lambda_2 = 0$$
Or, if the vector set is redundant, then it is linearly dependent
