A linear mapping is a function that transforms vectors in a way that respects the two fundamental vector operations: addition and scalar multiplication. 
In simpler terms, if you add two vectors and then transform them, you get the same result as if you transform them first and then add the results. The same holds true for scaling a vector

### What is a "transform"?

Think of a "transformation" or "mapping" as a **rule** or a **function** that takes an input vector and produces an output vector. It can change the vector's length, direction, or both.

You asked if it's just scaling. Scaling is one _type_ of transformation, but there are others. Imagine a vector `v = (2, 1)`. We can apply different rules to it:

- **Scaling** by 2 would transform it to `(4, 2)`.
    
- **Rotating** it 90 degrees counter-clockwise would transform it to `(-1, 2)`.
    
- **Reflecting** it across the y-axis would transform it to `(-2, 1)`.
    

All of these are different transformations. The "linear" part of a "linear transformation" refers to a special kind of transformation that follows specific rules.1

### What does "preserving the grid structure" mean?

This is the most important idea. You're right to think visually. A linear transformation can stretch, shrink, or rotate the grid, but it has two strict rules:

1. All grid lines must **remain parallel** and **evenly spaced**.
	`Φ(v_1 + v_2) = Φ(v_1) + Φ(v_2)`:
		Φ = function that transforms the space
		this states that
		 if the  the _linear combination of original vectors_ converted into transformed vector is equal to the _linear combination of transformed vectors_ then the transformed vector space is has parallel lines<img src ='licensed-image.jpeg' style="filter: invert(1);">
	The rule `Φ(x + y) = Φ(x) + Φ(y)` is a way to compare two different paths to the same destination:
	
	- **Left Side `Φ(x + y)`**: First, add `x` and `y` to get the diagonal vector. Then, apply the transformation `Φ` to that single resulting vector.
    
	- **Right Side `Φ(x) + Φ(y)`**: First, transform `x` and `y` individually. Then, add those two _new_ vectors together.
    
2. The **origin (0,0) must not move**.
