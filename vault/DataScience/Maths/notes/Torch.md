### 🔹 **What is `torch`?**

`torch` is the **core PyTorch library** — it provides:

- The **tensor object** (like NumPy arrays, but with GPU support).
	[[Tensor]]
    
- Mathematical operations on tensors (linear algebra, statistics, etc.).
	[[Tensor]]
	
- Automatic differentiation (`torch.autograd`) to compute gradients.
    [[autograd]]
    
- Building blocks for deep learning (`torch.nn`, `torch.optim`, etc.).
	[[Torch.nn]]
	[[Torch.optim]]
	
- others
	[[Data_Loader]]
---

### 🔹 In one line:

**`torch` = a scientific computing and machine learning library that uses tensors + autograd to build and train neural networks (on CPU or GPU).**

---

👉 Think of it as **NumPy + GPU + autodiff + neural net tools**.
