groups  is a set of element where each element could perform single operation with each other and fulfill this criteria:

>_consider all elements in  examples are  of set Real number_

[[Associative]] : the way you group the element's operations doesn't change the result.
eg:  1+(9+5) ==  1+(9+5). 

[[closure]] : output of the operations should belong from the set itself
eg : 2+2 = 4 (4 is from real numbers)

neutral element : it is an element from the set itself, which has property of not changing the output when it is applied to any other element of from the set
eg: 4+0 = 0 
	here:  0 is an neutral element.

negative element: each an element from the set should consist an element which undo itself (bring it to neutral element)
eg: 4 - 4 =0  
	here  : 4 has -4 that brings it to neutral element 0 

> since Real-number with addition operation fulfills all the criteria it could be called as group (R , +)
