# Absolute Errors
 - it is the difference in true value and measured value
 $$ \text{Absolute value} =|\text{true value} - \text{measured value}|$$
 - eg, 
	 - $\text{measured value} = 10$
	 - $\text{true Value} = 11$
	 - $1\text{ is the absolute value of a}$



--- 
# relative Errors
It is the ratio of measured value and true value
$$ \text{Relative Errors} = \frac{ \text{measured value}}{\text{true value}}$$

- since it is a ratio, it allow us to compare errors with different unit
---

# percentage Errors
 give us the ratio of error at the scale of hundred
$$\text{percentage Errors}= \text{Relative Errors}\times 100$$
- easy to grasp the magnitude of error
