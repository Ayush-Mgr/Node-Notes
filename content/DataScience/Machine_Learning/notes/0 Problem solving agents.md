It is a system which perceives environment search for the problem. Look for its current state and find the possible actions with minimal costs to  get at the goal state(solution)

 - **$S$**: Represents the set of all possible states, including the start state and goal states.
  - **$A$**: Denotes the set of actions or rules applicable to the states.
  - **$action(S)$**: Specifies the actions available for a given state $S$.
  - **$result(s, a)$**: Indicates the state resulting from applying action $a$ in state $s$.
  - **$cost(s, a)$**: Represents the cost associated with performing action $a$ in state $s$.
