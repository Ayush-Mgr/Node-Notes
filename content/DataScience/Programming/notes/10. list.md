### 🔹 **Introduction to List**

A **list** is a collection of items in a single variable. Lists are **ordered**, **changeable**, and can contain **duplicates**.

**Example:**

```python
fruits = ["apple", "banana", "cherry"]
```

---

### 🔹 **List Operations**

Basic operations you can do on lists:

- **Concatenation** (`+`)
    
- **Repetition** (`*`)
    
- **Membership** (`in`)
    
- **Indexing** (`[]`)
    
- **Slicing** (`[:]`)
    

**Example:**

```python
a = [1, 2]
b = [3, 4]
print(a + b)       # [1, 2, 3, 4]
print(a * 2)       # [1, 2, 1, 2]
print(2 in a)      # True
print(a[1])        # 2
print(b[0:2])      # [3, 4] - list[start:stop:step]
```

---

### 🔹 **Traversing a List**

Going through each element in a list.

**Example (using for loop):**

```python
names = ["Alice", "Bob", "Charlie"]
for name in names:
    print(name)
```

---

### 🔹 **List Methods and Built-in Functions**

**Common Methods:**

- `append()` – Add item at end
    
- `insert()` – Add item at index
    
- `remove()` – Remove item by value
    
- `pop()` – Remove item by index
    
- `sort()` – Sort list, _(all elements in the list should be of the same data type)_
    
- `reverse()` – Reverse list
    

**Built-in Functions:**

- `len()` – Number of elements
    
- `min()`, `max()`, `sum()`
    

**Example:**

```python
nums = [3, 1, 4]
nums.append(2)       # [3, 1, 4, 2]
print(len(nums))     # 4
nums.sort()
print(nums)          # [1, 2, 3, 4]
```

---

### 🔹 **List Manipulation**

Modifying the list: adding, removing, updating items.

**Example:**

```python
colors = ["red", "blue", "green"]
colors[1] = "yellow"     # update
colors.append("purple")  # add
del colors[0]            # delete
print(colors)            # ['yellow', 'green', 'purple']
```
