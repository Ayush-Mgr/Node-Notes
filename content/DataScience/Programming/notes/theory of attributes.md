- positive Attributes or frequencies denoted as $(A,B,C,D....)$ 
- negative Attributes or frequencies denoted as $( \bar A,\bar B,\bar C,\bar D ....)$
- Total Data = N  
- attribute of data = N(A)  
## Consistency of Data  
- **Rules to be data consistence**  
	- $N$ = sum of all frequency class  
		- $N=N(A,\bar B)+N(\bar A,B)+N(A,B)+N(\bar A,\bar B)$
	- Logical limits of frequency (sub class are always less or equal then their super set )  
		- $N(A) >= N(A,B) \& N(A,\bar B) <= N(A)$  
	- Multi attribute Data :  
		- when more attributes are involved combination of all attributes should be = overall total  
## Independence and Association of Attributes  
- related (associated) or unrelated (independent) Attributes  
- **independence** (when the occurrence of $N(A)$ don't effect $N(B))$  
    -  mathematically denoted as :
	    - $P(A ∩ B)=P(A).P(B)$
	    - eg. [[theory of attributes  example 1]]
