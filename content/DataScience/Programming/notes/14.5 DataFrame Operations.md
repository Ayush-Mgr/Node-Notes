## ✅ **Other Useful DataFrame Operations**

### 🔹 1. **Filtering / Conditional Selection**

Select rows that meet certain conditions.

```python
df[df['Age'] > 30]
```

---

### 🔹 2. **Adding / Modifying Columns**

```python
df['New_Col'] = df['Col1'] + df['Col2']
```

---

### 🔹 3. **Renaming Columns**

```python
df.rename(columns={'old': 'new'}, inplace=True)
```

---

### 🔹 4. **Dropping Columns or Rows**

```python
df.drop('column_name', axis=1, inplace=True)  # Drop column
df.drop(0, axis=0, inplace=True)              # Drop row by index
```

---

### 🔹 5. **Data Type Conversion**

```python
df['Age'] = df['Age'].astype(float)
```

---

### 🔹 6. **Applying Functions with `apply()`**

```python
df['Col'] = df['Col'].apply(lambda x: x * 2)
```

---

### 🔹 7. **Merging / Joining DataFrames**

```python
pd.merge(df1, df2, on='common_column')
```

---

### 🔹 8. **Concatenating DataFrames**

```python
pd.concat([df1, df2], axis=0)  # Stack vertically
```

---

### 🔹 9. **Pivot Tables**

```python
df.pivot_table(values='Sales', index='Region', aggfunc='sum')
```

---

### 🔹 10. **Value Counts / Unique Values**

```python
df['City'].value_counts()
df['City'].unique()
```
