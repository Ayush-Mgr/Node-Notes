## types of SQL command:

1. Data definition language (DDL)
2. Data manipulation language (DML)
3. Data control language(DCL)
4. Traction control language (TCL)

Data definition language (DDL):
 - they are auto committed/the day changes are permanently  saved in the database
 - types of DLR:
	 - [[0.1 Creating a DATABASE process|create]]
	 - [[0.1 Creating a DATABASE process|alter]]
	 - [[0.1 Creating a DATABASE process|Drop]]
	 - [[0.1 Creating a DATABASE process|truncate]]  

Data manipulation language(DML):
 - they are not Auto commented, the changes are not prominently saved in the database
 - they are used for manipulating the existing data
 - types of DML  :
	 - [[0.1 Creating a DATABASE process|select]] (also called the data query language DQL)
	 - [[0.1 Creating a DATABASE process|update]]
	 - [[DELETE|delete]]
	 - [[0.1 Creating a DATABASE process||insert]]

Data control language (DCL):
 - where are used to give or reject the authority of data?
 - types of DCL:
	- Grant
	- revoke

Traction control language (TCL):
 - it is used to manage the database transactions 
 - types of TCL:
	- [[installation|commit]]
	- [[installation|rollback]]
