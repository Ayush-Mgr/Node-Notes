## straight line
- geometrical representation of a straight line in 2D plane : $Ax+By+c=0$ 
	- where $A, B, C$ are constants 
	- if point is given $(6,7)$ 
	- $x=6 \text { and } y=7$
	- replacing the values if LHS = RHS(0) the line cross the point $(6,7)$ 
- slope-intercept form **$y=a+bx$**
	-  **Steps for Conversion**
		1. Start with the standard equation:
		    
		    $Ax+By+C=0$
		2. Rearrange to isolate $y$ on one side:
		    
		    $By=−Ax−C$
		3. Divide through by B (assuming B≠0B=0):
		    
		    $y=(−A/B)x + (−C/B)$
	- $a$ = $(−C/B)$  , represents the $y$ intercept  , The **y-intercept** is the point where the line crosses the $y$-axis. At this point, $x=0$
	- b =  $(−A/B)$ , represents the slope of the data , the amount of change( + or - or 0 ) in the y as x increase   
	![[Pasted image 20241211113354.png]]
	- if $(x_1,y_1),(x_2,y_2)$ given then $b=\frac{y_1-y_2}{x_1-x_2}$
	-  if the line pass through the point $(x_0, y_0)$ which means *any* point at line and has slope $b$ then $y- y_0 = b(x - x_0)$
##  Parabola
- geometrical representation 
	- $y=a+bx+cx^2$
	- $a,b,c$  are constant 
	- $x^2$ should not be 0  , $c \not= 0$
- represents the curve of flying object , going up and down
