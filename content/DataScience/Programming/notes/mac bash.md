### **Mac Bash Commands Cheat Sheet**  
📌 Quick reference for essential macOS Terminal commands.

#### 📂 **File & Directory Management**  
```bash
ls -la            # List all files (including hidden ones)
cd foldername     # Change directory
pwd               # Show current directory path
mkdir newfolder   # Create a new folder
rmdir foldername  # Remove empty folder
rm -r foldername  # Delete folder and its contents
```

#### 📄 **File Operations**  
```bash
touch file.txt          # Create a new file
echo "Hello" > file.txt # Create/write text to a file
echo "More" >> file.txt # Append text to a file
cat file.txt            # View file contents
mv file.txt newname.txt # Rename/move a file
cp file.txt copy.txt    # Copy a file
rm file.txt             # Delete a file
```

#### ⚙️ **Permissions & Processes**  
```bash
chmod +x script.sh    # Make a file executable
chown user file.txt   # Change file owner
ps aux                # Show running processes
kill -9 PID          # Kill a process by PID
```

#### 🔍 **Searching & System Info**  
```bash
find . -name "file.txt"   # Find a file
grep "word" file.txt      # Search inside a file
top                      # Show system resource usage
df -h                    # Check disk space
```

#### ⏳ **Shortcuts**  
```bash
CTRL + C  # Stop a command  
CTRL + L  # Clear Terminal  
CTRL + D  # Exit Terminal  
```
