### 🔹 **Introduction to Dictionaries**

A **dictionary** is a collection of key-value pairs.  
It is **unordered**, **changeable**, and **doesn't allow duplicate keys**.

**Syntax:**

```python
student = {"name": "Alice", "age": 20, "grade": "A"}
```

---

### 🔹 **Traversing a Dictionary**

Going through keys, values, or both.

**Example:**

```python
info = {"name": "Tom", "age": 18}

# Traverse keys
for key in info:
    print(key)

# Traverse values
for value in info.values():
    print(value)

# Traverse key-value pairs
for key, value in info.items():
    print(key, ":", value)
```

---

### 🔹 **Dictionary Methods and Built-in Functions**

**Common Methods:**

- `keys()` – Returns all keys
    
- `values()` – Returns all values
    
- `items()` – Returns key-value pairs
    
- `get()` – Get value by key (safe)
    
- `update()` – Add/update items
	 1. Using a dictionary:
	
	```python
	
	info.update({"name": "Jerry"})
	
	```
	
	 2. Using keyword arguments:
	
	```python
	
	info.update(name="Jerry")
	
	```
	
	 3. Direct assignment:
	
	```python
	
	info["name"] = "Jerry"
	
	```
	   
- `pop()` – Remove item by key
    

**Built-in Functions:**

- `len()` – Number of items
    
- `str()` – Convert to string
    
- `type()` – Check type
    

**Example:**

```python
person = {"name": "Eve", "city": "Delhi"}
print(len(person))           # 2
print(person.get("name"))    # Eve
person.update({"age": 25})   # Add new item
print(person)
```

---

### 🔹 **Manipulating Dictionaries**

Changing, adding, or deleting key-value pairs.

**Example:**

```python
car = {"brand": "Toyota", "year": 2020}

# Update value
car["year"] = 2023

# Add new key-value pair
car["color"] = "blue"

# Delete a key
del car["brand"]

print(car)  # {'year': 2023, 'color': 'blue'}
```
