- "Art of reading between lines of table"  , Interpolation is the process of finding the value of _unknown_  $y$ for a given value of $x$ (which was not in data set ) within a range of _known_ data points $(x_i​,y_i​)$. It involves estimating the value of the function at a point that lies between the known data points.
