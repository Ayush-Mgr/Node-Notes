**for Loop:**  
Used to repeat a block of code for each item in a sequence.  
👉 Example:

```python
for i in range(3):  
    print(i)  
```

**while Loop:**  
Repeats a block of code as long as a condition is true.  
👉 Example:

```python
i = 0  
while i < 3:  
    print(i)  
    i += 1  
```

**Nested Loop:**  
A loop inside another loop to handle complex tasks.  
👉 Example:

```python
for i in range(2):  
    for j in range(2):  
        print(i, j)  
```
