### 🔹 **What is Splitting?**

**Splitting** means dividing one NumPy array into **multiple sub-arrays**.

---

### 🔹 **1. Splitting 1D Arrays**

Use `np.split()` or `np.array_split()`.

```python
import numpy as np

a = np.array([10, 20, 30, 40, 50, 60])

# Split into 3 equal parts
print(np.split(a, 3))
# Output: [array([10, 20]), array([30, 40]), array([50, 60])]
```

✅ Use `np.array_split()` when sizes are **not equal**:

```python
print(np.array_split(a, 4))
# Output: [array([10, 20]), array([30, 40]), array([50]), array([60])]
```

---

### 🔹 **2. Splitting 2D Arrays**

```python
b = np.array([[1, 2, 3, 4],
              [5, 6, 7, 8]])

# Split into 2 equal horizontal parts (columns)
print(np.hsplit(b, 2))
# Output: [array([[1, 2], [5, 6]]), array([[3, 4], [7, 8]])]

# Split into 2 vertical parts (rows)
print(np.vsplit(b, 2))
# Output: [array([[1, 2, 3, 4]]), array([[5, 6, 7, 8]])]
```

---

### 🔹 **Useful Split Functions**

|Function|Purpose|
|---|---|
|`np.split()`|Equal parts only|
|`np.array_split()`|Unequal parts allowed|
|`np.hsplit()`|Horizontal (columns)|
|`np.vsplit()`|Vertical (rows)|
