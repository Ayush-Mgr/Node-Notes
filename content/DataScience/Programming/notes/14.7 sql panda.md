### 🔹 **1. Setup**

You need `pymysql` or `mysql-connector-python` and `sqlalchemy` installed:

```bash
pip install pymysql sqlalchemy
```

---

### 🔹 **2. Import Data from MySQL to Pandas**

```python
import pandas as pd
from sqlalchemy import create_engine

# Connect to MySQL (replace with your credentials)
engine = create_engine('mysql+pymysql://user:password@host/dbname')

# Read SQL query or table into DataFrame
df = pd.read_sql('SELECT * FROM tablename', con=engine)
print(df.head())
```

---

### 🔹 **3. Export Data from Pandas to MySQL**

```python
# Write DataFrame to MySQL table
df.to_sql('new_table', con=engine, if_exists='replace', index=False)
```

---

### ✅ Quick Tips:

- Use `if_exists='append'` to add data without deleting existing rows.
    
- Ensure your MySQL user has write permissions.
    
- For large data, consider batch inserts or CSV export/import.
