## **Trigger Structure (MySQL)**

```sql
delimiter //

CREATE TRIGGER trigger_name
{BEFORE | AFTER} {INSERT | UPDATE | DELETE}  -- timing & event
ON table_name                               -- table the trigger is attached to
FOR EACH ROW                                -- fires for every affected row
BEGIN
    -- SQL statements go here
	;
END//

delimiter;
```

### The “clauses” inside it are basically:

1. **Timing clause** → `BEFORE` or `AFTER`
    
2. **Event clause** → `INSERT`, `UPDATE`, or `DELETE`
    
3. **Target table** → `ON table_name`
    
4. **Execution scope** → `FOR EACH ROW`
    
5. **Body** → SQL code between `BEGIN` and `END` that does the action
    

---

### **Example**

```sql
CREATE TRIGGER prevent_low_price
BEFORE INSERT ON Sales_Table
FOR EACH ROW
BEGIN
    IF NEW.price < 1 THEN
        SIGNAL SQLSTATE '45000'
        SET MESSAGE_TEXT = 'Price must be at least 1';
    END IF;
END;
```

- **BEFORE INSERT** → runs before new row is added
    
- **NEW.price** → refers to the incoming value
    
- `SIGNAL` → throws an error to stop the insert
