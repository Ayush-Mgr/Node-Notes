### 🔹 **What is Loading Arrays?**

It means **reading data** stored in external files (like `.txt`, `.csv`, `.npy`) into NumPy arrays.

---

### 🔹 **1. Loading from Text Files (.txt or .csv)**

Use: `np.loadtxt()` or `np.genfromtxt()`

**Example (CSV or space-separated values):**

```python
import numpy as np

data = np.loadtxt('data.txt')      # Loads numeric data
print(data)
```

> ✅ Add arguments if needed:

```python
data = np.loadtxt('data.csv', delimiter=',')
```

---

### 🔹 **2. Loading from `.npy` Files** (NumPy's own binary format)

Use: `np.load()`

**Example:**

```python
# First, save an array (for reference)
np.save('array_file.npy', np.array([1, 2, 3]))

# Now load it
loaded_array = np.load('array_file.npy')
print(loaded_array)     # [1 2 3]
```

---

### 🔹 **3. Loading from `.npz` Files** (Compressed multiple arrays)

```python
# Save multiple arrays
np.savez('multi_arrays.npz', x=[1, 2], y=[3, 4])

# Load them
arrays = np.load('multi_arrays.npz')
print(arrays['x'])      # [1 2]
print(arrays['y'])      # [3 4]
```

---

### ✅ Quick Tips

|Function|Use for|
|---|---|
|`loadtxt()`|Simple text/CSV data|
|`genfromtxt()`|Text with missing values|
|`load()`|`.npy` or `.npz` files|
