## 🔹 **Introduction to Descriptive Statistics (in Pandas)**

Descriptive statistics summarize data to understand patterns and distributions.

Use `.describe()` or individual methods like `.mean()`, `.median()`, `.std()`, etc.

### ✅ **Example:**

```python
import pandas as pd

data = {'Marks': [80, 90, 85, 70, 95]}
df = pd.DataFrame(data)

print(df.describe())
```

**Output:**

```
         Marks
count   5.000000
mean   84.000000
std     9.617692
min    70.000000
25%    80.000000
50%    85.000000
75%    90.000000
max    95.000000
```

### 🔹 Individual Functions

```python
df['Marks'].mean()     # Average
df['Marks'].median()   # Middle value
df['Marks'].std()      # Standard deviation
df['Marks'].min()      # Lowest mark
df['Marks'].max()      # Highest mark
```

## 🔹 **Data Aggregations in Pandas**

Aggregation means applying **summary functions** like `sum()`, `count()`, `mean()`, etc., on groups of data.

### ✅ **Using `.groupby()`**

`groupby()` is used to **group data** by one or more columns and then apply **aggregation functions** like `sum()`, `mean()`, `count()`, etc.

### 🔹 **Syntax:**

```python
df.groupby('column_name').function()
```

### ✅ **Example:**

```python
import pandas as pd

data = {'Department': ['HR', 'IT', 'HR', 'IT'],
        'Salary': [30000, 50000, 35000, 60000]}

df = pd.DataFrame(data)

# Group by Department and get average salary
df.groupby('Department')['Salary'].mean()
```

```python
df.groupby('Class')['Marks'].agg(['mean', 'min', 'max'])
```
