Yes, $P(X = x)$ is referred to as the **Probability Mass Function (PMF)** for discrete [[10. Random variable|random variables]].

### Key Points:
1. **Definition**:
   - Mathematically: $P(X = x) = f(x)$, where $f(x)$ is the PMF.

2. **Properties**:
   - $P(X = x) \geq 0$ for all $x$ (probabilities are non-negative).
   - The sum of the probabilities over all possible values of $X$ is 1:
     $$
     \sum_{x \in S} P(X = x) = 1,
    $$
     where $S$ is the set of all possible values of $X$.

3. **Discrete Random Variables**:
   - The PMF is only defined for **discrete** random variables, which take on a countable set of values.

4. **Examples**:
   - For a fair six-sided die, the PMF for $X$, the result of the roll, is:
     $$
     P(X = x) = \begin{cases} 
     \frac{1}{6}, & x \in \{1, 2, 3, 4, 5, 6\} \\
     0, & \text{otherwise.}
     \end{cases}
    $$
